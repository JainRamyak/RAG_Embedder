WHAT THIS PROJECT IS
================================================================
A production-style "Ask My Docs" RAG system.
Ingest any folder of documents (txt, md, PDF) and ask questions
in plain English. Returns cited answers like:
  "Python was created by Guido van Rossum [1]."
Answers ONLY from your documents. Exposed as a REST API.
Supports 3 LLM backends: Gemini (free), OpenAI, Anthropic.

================================================================
ARCHITECTURE
================================================================
HTTP Request
    │
    ▼
[api/main.py] FastAPI
    GET  /health   — liveness check
    POST /ingest   — ingest a directory of documents
    POST /query    — ask a question, get cited answer
    GET  /docs     — Swagger UI (browser)
    │
    ▼
[src/pipeline.py] AskMyDocsPipeline
    │
    ├── INGEST:
    │     loader.py      → .txt / .md / .pdf
    │     chunker.py     → 512-char overlapping chunks
    │     embedder/      → 384-dim vectors
    │     vec_store      → FAISS (semantic search)
    │     bm25_store     → BM25 (keyword search)
    │
    └── ASK:
          hybrid_retriever.py  → FAISS + BM25 via RRF
          reranker.py          → cross-encoder reranking
          answer_chain.py      → LLM with citation prompt
          → {"answer": "...[1]...", "sources": [...]}

LLM PROVIDER SWITCHING
================================================================
Change ONE line in .env to switch providers:

  LLM_PROVIDER=gemini       ← FREE, recommended, no credit card
  LLM_PROVIDER=openai       ← needs credits at platform.openai.com
  LLM_PROVIDER=anthropic    ← needs credits at console.anthropic.com

Provider details:
  gemini:
    Model:   gemini-2.5-flash
    SDK:     google-genai==1.73.1
    Cost:    Free (1000 req/day on free tier)
    Key:     GEMINI_API_KEY (get at aistudio.google.com)

  openai:
    Model:   gpt-4o-mini
    SDK:     openai==2.32.0
    Cost:    ~$0.15 per 1M input tokens (paid)
    Key:     OPENAI_API_KEY (get at platform.openai.com)
    Minimum: $5 credit deposit required

  anthropic:
    Model:   claude-sonnet-4-20250514
    SDK:     anthropic==0.94.1
    Cost:    ~$3 per 1M input tokens (paid)
    Key:     ANTHROPIC_API_KEY (get at console.anthropic.com)
    Minimum: $5 credit deposit required

Verify after switching:
  python3 -c "from config import settings; \
  print(settings.llm_provider, bool(settings.gemini_api_key))"

KEY DECISIONS
================================================================
- Default LLM: gemini-2.5-flash (free, no credit card needed)
- google-generativeai REMOVED (incompatible with Python 3.14)
- google-genai==1.73.1 used instead (pure Python, works on 3.14)
- gemini-2.0-flash has limit=0 on free tier — use gemini-2.5-flash
- p.vec_store NOT p.vector_store (pipeline attribute name)
- loader function: load_documents() NOT load_directory()
- api/__init__.py must be EMPTY (not contain any code)
- load_dotenv(override=True) in config.py
- LLM client created INSIDE functions (lazy init, not at import)
- _optional() must be defined BEFORE _Settings class in config.py
- Never commit .env, Key.txt, or tokens of any kind
- GitHub push needs Personal Access Token (not password):
  git push https://JainRamyak:TOKEN@github.com/JainRamyak/RAG_Embedder.git main
- 503 from Gemini = temporary server overload, just re-run
