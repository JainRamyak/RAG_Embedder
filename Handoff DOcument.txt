================================================================
WHAT THIS PROJECT IS
================================================================
A production-style "Ask My Docs" RAG system.
Ingest any folder of documents (txt, md, PDF) and ask questions
in plain English. Returns cited answers like:
  "Python was created by Guido van Rossum [1]."
Answers ONLY from your documents. Exposed as a REST API.
Supports 4 LLM backends switchable via one .env line.

================================================================

ARCHITECTURE
================================================================
HTTP Request
    │
    ▼
[api/main.py] FastAPI
    GET  /health   — liveness check
    POST /ingest   — ingest a directory of documents
    POST /query    — ask a question, get cited answer
    GET  /docs     — Swagger UI (browser)
    │
    ▼
[src/pipeline.py] AskMyDocsPipeline
    │
    ├── INGEST:
    │     loader.py      → .txt / .md / .pdf
    │     chunker.py     → 512-char overlapping chunks
    │     embedder/      → 384-dim vectors (all-MiniLM-L6-v2)
    │     vec_store      → FAISS (semantic search)
    │     bm25_store     → BM25 (keyword search)
    │
    └── ASK:
          hybrid_retriever.py  → FAISS + BM25 fused via RRF
          reranker.py          → cross-encoder reranking
          answer_chain.py      → gemini/mistral/openai/anthropic
          → {"answer": "...[1]...", "sources": [...]}

================================================================
KEY DECISIONS
================================================================
- Default LLM: gemini-2.5-flash (free, no credit card)
- Mistral SDK: mistralai==2.4.2 uses Mistral() not MistralClient()
  and client.chat.complete() not client.chat()
  (old v1 imports will crash — always use v2 API)
- google-generativeai REMOVED (Python 3.14 incompatible)
- google-genai==1.73.1 used instead (pure Python)
- gemini-2.0-flash has limit=0 — use gemini-2.5-flash
- p.vec_store NOT p.vector_store
- load_documents() NOT load_directory()
- api/__init__.py must be EMPTY
- load_dotenv(override=True) in config.py
- LLM client created INSIDE functions (lazy init)
- _optional() must be BEFORE _Settings class in config.py
- config.py default LLM_PROVIDER changed from anthropic → gemini
- Never commit .env, Key.txt, or tokens
- GitHub push: use Personal Access Token
  git push https://JainRamyak:TOKEN@github.com/JainRamyak/RAG_Embedder.git main
- 503 from Gemini = temporary overload, re-run
- 429 from OpenAI = no credits, switch to gemini or mistral
